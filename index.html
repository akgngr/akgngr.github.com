Hakkımda İletişim

KISACA BEN..!

10 yıldır bilgisayar teknolojilerine ilgilenmekteyim, özellikle web teknolojilerine ayrı bir ilgim var. İlk başlarda web tasarım ile ilgilendim. HTML ve CSS ile statik web siteleri yapmaya başladım. Kullanıcılarla etkileşimli sayfalar yapmak için PHP dilini öğrendim. Daha sonra hızlı bir şekilde web sayfaları yapmak için içerik yönetim sistemleri üzerine yoğunlaştım, ilk öğrendiğim içerik yönetim sistemi JOOMLA idi, JOOMLA ile web siteleri yaptım. Daha sonra WORDPRESS öğrendim ve kullanmaya başladım. Bir çok içerik yönetim sistemlerini kullandım ve en sonunda DRUPAL’i keşfettim. Drupal ile her türden web siteleri yapma konusunda kendimi geliştirdim. Tanıtım sayfaları, emlak sayfaları, alış veriş sayfaları, blog sayfaları gibi bir çok alanda web sayfaları yaptım. Hazır drupal temaları çok basit kaçtığı için drupale nasıl HTML temaları giydiririm düşüncesiyle drupale tema giydirmeyi öğrenerek hazır HTML temaları ve yaptığım temaları drupale giydirmeye başladım. Son 2 yıldır Laravel Framework ile Symfony Framework kullanmayı öğrendim ve web uygulamaları geliştirmeye başladım. Suanda kendimi Symfony 4 Framework bilgimi geliştirip web sayfaları ve web uygulamaları yapmaktayım.
